## Кэшировальщик по 2check! :robot:	
Кэширует запросы в Redis. Специальный Middleware проверяет существование объектов в Redis.
Если объектов нет - блокирует Middleware для всех остальных по данному объекту и берет данные из базы в контроллере.
Затем, проходя назад через пайплайн :leftwards_arrow_with_hook:, проходит через этот милдвейр, где заносит все данные в Redis, и разблокирует его для остальных.

Другие входящие в это время запросы, проверяют Redis, и если не нахолят там ничего, начинают ожидать снятие блокировки. 
После, ещё раз проверяют наличие данных в Redis.

## Запуск :dvd::

1. Развертывание:
	1. Скачайте PostgreSQL, Redis на ПК 
	2. Скачайте Docker Desktop. Затем запустите либо через Visual Studio, либо пропишите команду
		``` 
		docker-compose up -d
		```
2. Запуск
	1. База данных
		1. Мигрировать: в консоли диспетчера пакетов сделать:
			```
			Add-Migration My
			Update-Database
			```
		2. В коде PackageContext раскомментировать указанную там строку 
	2. Для создания данных в базе данных:
		1. `POST api/Packages/ensure`
		2. Отправить ensure в Swagger
	3. Как пример теста, отправить table = body, filter = Strong. Если возвращает 200 - это контроллер из базы, 201 - middleware из Redis
